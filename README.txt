
		心常乐数独小游戏 简介
		
======================

    本软件是一款windows平台下的数独游戏。 
    
    本软件是开源、免费软件。
    具体版权说明见COPYING.txt。

    本软件的编译方法见Build.txt。
======================

主要的版本历史：

2013年10月，1.0发布(采用CodeBlocks+MinGW+Wxwidgets开发)
2014年4月， 1.0.2发布(将软件修改为通过MinGW及纯C语言实现，同时软件走向开源)
